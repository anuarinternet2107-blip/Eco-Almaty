# Инструкция по получению API ключа, чтобы ИИ работал:

Зайдите на сайт aistudio.google.com
Войдите через Google аккаунт
Нажмите "Get API key" в левом боковом меню
Нажмите "Create API key"
Выберите "Create API key in new project"
Ключ сгенерируется и вы должны его скопировать
В папке проекта создайте файл и назовите его = .env
Внутри .env напишите одну строку:

GEMINI_API_KEY=ваш ключ

без пробелов. 

Структура должна выглядеть вот так:

```
├── app.py              # Сервер и API
├── database.db         # База данных (создаётся автоматически)
├── templates/
│   ├── index.html      # Страница входа / регистрации
│   └── map.html        # Карта
├── static/
│   └── LeafLogo.png    # Логотип
├── uploads/            # Загруженные фотографии
└── .env                # API ключ
```

# Как запустить сайт
В верхней стороне экрана нажмите на Terminal, потом на New Terminal
После активации и загрузки терминала введите эти команды:

# Windows
python -m pip install flask google-genai pillow python-dotenv

# Mac / Linux
pip3 install flask google-genai pillow python-dotenv

# После того как скачаются нужные библиотеки запустите эти команды

# Windows
python app.py

# Mac / Linux
python3 app.py

В терминале должно появиться что то похожее на 

Running on http://...

скопируйте ссылку и вставьте в гугл
